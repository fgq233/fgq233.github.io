### JQuery 选择器
### 一、基本选择器

| 选择器        | 功能                   | 返回值   | 示例   |
| :----------  | :---------------------| :------ |:------ |
| `#id`       | 根据id匹配一个元素        | 单个元素 | `$("#id")` |
| `element`   | 根据元素名匹配所有元素     | 元素集合 | `$("div")` |
| `.class`    | 根据样式类名匹配所有元素   | 元素集合 | `$("cls")` |
| `*`         | 匹配所有元素             | 元素集合 | `$("*")` |
| `selector1, selector2 ...` | 将每个选择器匹配到的元素合并后返回 | 元素集合 | `$("#id, div, .cls")` |

### 二、层次选择器

| 选择器 | 功能 | 返回值 | 相同效果方法 |
| :--- | :--- | :------ | :------ |
| `element1 空格 element2` | 祖先元素所有子孙  | 元素集合 |   | 
| `element1 > element2`   | 父亲元素所有儿子  | 元素集合 |   | 
| `element1 + element2`   | 后面的下一个兄弟元素     | 元素集合 | `next()` | 
| `element1 ~ element2`   | 后面的所有兄弟元素     | 元素集合 |  `nextAll()` | 


```
<ul id="ul">
    <li id="li1"></li>
    <li id="li2"></li>
    <li id="li3">
        <ul>
            <li id="li3_1"></li>
            <li id="li3_2"></li>
        </ul>
    </li>
    <li id="li4"></li>
</ul>

console.log($("#ul li").length);    // 6
console.log($("#ul > li").length);  // 4
// 后面下一个兄弟
console.log($("#li2 + li").length);      // 1   (li3)
console.log($("#li2").next().length);    // 1
// 后面的所有兄弟
console.log($("#li2 ~ li").length);        // 2   (li3、li4)
console.log($("#li2").nextAll().length);   // 2
// 所有同级兄弟
console.log($("#li2").siblings().length);  // 3   (li1、li3、li4)
```




### 三、表单选择器

| 选择器 | 功能 | 返回值 | 
| :--- | :--- | :------ |
| `:input` | 所有input、textarea、select  | 元素集合 | 
| `:text` | 所有单行文本框  | 元素集合 | 
| `:password` |  所有密码框  | 元素集合 | 
| `:radio` | 所有单选按钮  | 元素集合 | 
| `:checkbox` | 所有复选框  | 元素集合 | 
| `:submit` | 所有提交按钮  | 元素集合 | 
| `:image` | 所有图像域  | 元素集合 | 
| `:reset` | 所有重置按钮  | 元素集合 | 
| `:button` | 所有按钮  | 元素集合 | 
| `:file` | 所有文件域  | 元素集合 | 


```
<form id="form1">
    <textarea></textarea>   <br>
    <select>
        <option value="1">666</option>
        <option value="2">888</option>
    </select><br>
    <input type="text" value="text"/><br>
    <input type="password" value="666"/><br>
    <input type="radio" value="1"/><br>
    <input type="checkbox" value="1"/><br>
    <input type="submit" value="提交"/><br>
    <input type="image" src="https://fgq233.github.io/imgs/js/js.jpg" alt="图像形式提交按钮"/><br>
    <input type="reset" value="重置"/><br>
    <input type="button" value="按钮"/><br>
    <input type="file"/><br>
</form>

console.log($("#form1 :input").length);     // 11
console.log($("#form1 :text").length);      // 1
console.log($("#form1 :password").length);  // 1
console.log($("#form1 :radio").length);     // 1
console.log($("#form1 :checkbox").length);  // 1
console.log($("#form1 :submit").length);    // 1
console.log($("#form1 :image").length);     // 1
console.log($("#form1 :reset").length);     // 1
console.log($("#form1 :button").length);    // 1
console.log($("#form1 :file").length);      // 1
```



### 四、过滤选择器
* 过滤选择器根据某类规则过滤，写法大部分以冒号（`:`）开头

#### 1. 索引过滤
索引从0开始

| 选择器 | 功能 | 返回值 | 相同效果方法 |
| :--- | :--- | :------ | :------ |
| `:first` | 第一个元素  | 单个元素 | `first()` | 
| `:last` | 最后一个元素  | 单个元素 | `last()` | 
| `:not(selector)` | 除了给定元素外的所有元素  | 元素集合 |  | 
| `:even` | 索引值为偶数的所有元素  | 元素集合 | |
| `:odd` | 索引值为奇数的所有元素  | 元素集合 | |
| `:eq(index)` | 等于给定索引值的元素 | 单个元素 | |
| `:gt(index)` | 大于给定索引值的所有元素  | 元素集合 | |
| `:lt(index)` | 小于给定索引值的所有元素  | 元素集合 | |

```
<ul id="ul">
    <li>第一行</li>
    <li>第二行</li>
    <li>第三行</li>
    <li>第四行</li>
    <li>第五行</li>
</ul>

console.log($("li:first").html());      // 第一行
console.log($("li").first().html());   // 第一行

console.log($("li:last").html());       // 第五行
console.log($("li").last().html());   // 第五行

console.log($("li:even").length);   // 3
console.log($("li:odd").length);    // 2

console.log($("li:eq(3)").html());  // 第四行
console.log($("li:not(:eq(3))").length);  // 4
console.log($("li:lt(3)").length);       // 3
console.log($("li:gt(3)").length);      // 1
```


#### 2. 内容过滤
根据元素中的文字内容或所包含子元素的特征过滤

| 选择器 | 功能 | 返回值 | 
| :--- | :--- | :------ |
| `:contains(文本)` | 包含给定文本的元素  | 元素集合 | 
| `:empty` | 不包含子元素或空文本的元素  | 元素集合 | 
| `:parent` | 包含子元素或有文本的元素  | 元素集合 | 
| `:has(selector)` | 匹配某个选择器的元素  | 元素集合 | 

```
<ul id="ul">
    <li>第一行</li>
    <li>第二行</li>
    <li>第三行</li>
    <li>第四行</li>
    <li>第五行</li>
</ul>

console.log($("li:first").html());  // 第一行
console.log($("li:last").html());   // 第五行

console.log($("li:even").length);   // 3
console.log($("li:odd").length);    // 2

console.log($("li:eq(3)").html());  // 第四行
console.log($("li:not(:eq(3))").length);  // 4
console.log($("li:lt(3)").length);       // 3
console.log($("li:gt(3)").length);      // 1
```



#### 3. 可见性过滤
根据元素是否可见过滤

| 选择器 | 功能 | 返回值 | 
| :--- | :--- | :------ |
| `:hidden` | 所有不可见元素，或者`type = "hidden"`的元素  | 元素集合 | 
| `:visible` | 所有可见元素  | 元素集合 | 

```
<ul id="ul">
    <li>A</li>
    <li style="display: none">B</li>
    <li>C</li>
    <li>D</li>
</ul>

console.log($("li:hidden").length);         // 1
console.log($("li:visible").length);        // 3
```


#### 4. 元素属性过滤
根据元素某个属性过滤元素，写法以`[`开始，以`]`结束

| 选择器 | 功能 | 返回值 | 
| :--- | :--- | :------ |
| `[attribute]` | 包含某个属性的元素  | 元素集合 | 
| `[attribute=value]` | 属性值 = value 的元素  | 元素集合 | 
| `[attribute!=value]` | 属性值 != value 的元素  | 元素集合 | 
| `[attribute^=value]` | 属性值 以value 开头的元素  | 元素集合 | 
| `[attribute$=value]` | 属性值 以value 结尾的元素  | 元素集合 | 
| `[selector1][selector2][selectorN]` | 满足多个条件的元素  | 元素集合 |

```
<span data-x="x1" title="666"></span>
<span data-x="x2" title="888"></span>
<span data-x="x3" title="666888"></span>
<span></span>

console.log($("span[data-x]").length);           // 3
console.log($("span[title='666']").length);      // 1
console.log($("span[title!='666']").length);     // 3
console.log($("span[title^='66']").length);      // 2
console.log($("span[title$='66']").length);      // 1
console.log($("span[title*='88']").length);      // 2
console.log($("span[data-x][title*='68']").length);   // 1
```



#### 5. 子元素过滤
获取父元素下指定的子元素

| 选择器 | 功能 | 返回值 | 
| :--- | :--- | :------ |
| `:first-child` | 每个父元素下第一个元素  | 元素集合 | 
| `:last-child` | 每个父元素下最后一个元素  | 元素集合 | 
| `:only-child` | 每个父元素下仅有的一个元素  | 元素集合 | 

```
<ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
</ul>
<ul>
    <li>4</li>
    <li>5</li>
    <li>6</li>
</ul>
<ul>
    <li>7</li>
</ul>

$("li:first-child").addClass("first");  // 1、4添加了 first样式
$("li:last-child").addClass("last");    // 3、6添加了last样式
$("li:only-child").addClass("only");    // 7 添加了only样式
```


#### 6. 表单对象属性过滤

| 选择器 | 功能 | 返回值 | 
| :--- | :--- | :------ |
| `:enabled` | 所有可用元素  | 元素集合 | 
| `:disabled` | 所有不可用元素  | 元素集合 | 
| `:checked` | 所有被`checked`选中的元素  | 元素集合 | 
| `:selected` | 所有被`selected`选中的`option`元素  | 元素集合 | 

```
<form id="form1">
    <input value="可用文本"/>
    <input value="不可用文本" disabled/>
    
    <input type="checkbox" value="0" checked />
    <input type="checkbox" value="1" />

    <select id="s">
        <option value="1">666</option>
        <option value="2">888</option>
    </select>
</form>

console.log($("input:enabled").length);         // 3
console.log($("input:disabled").length);        // 1
console.log($("input:checked").length);         // 1
console.log($("#s option:selected").text());    // 666
```
