### IDEA 中上传自己的项目到 Github
#### 1. 在github上创建一个远程仓库

#### 2. 方式一：控制台方式
* `git init`
* `git add .`
* `git commit -m 'first'`
* `git remote add origin 仓库地址`
* `git push origin master`

#### 3. 方式二：IDEA 中 UI 界面方式
* `VCS - Create Git Repository - 选中项目 - OK`
* 右键 `Git - Add`
* 菜单栏 `Git - Commit`
* 控制台 `git remote add origin 仓库地址`
* 控制台 `git push origin master`
