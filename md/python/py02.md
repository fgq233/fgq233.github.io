### 条件判断、循环
### 一、条件判断
```
if 条件:
    statement
  
if 条件:
    statement1
else:
    statement2
  
if 条件1:
    statement1
elif 条件2：
    statement2
else:
    statement3
```


### 二、循环
#### 1. while 循环
```
while 条件:
    循环体
    
while 条件:
    循环体
else:
    循环正常结束之后要执⾏的代码(非break结束)
```

#### 2. for 循环
```
for 临时变量 in 序列:
    循环体
    
for 临时变量 in 序列:
    循环体
else:
    循环正常结束之后要执⾏的代码(非break结束)
```


#### 3. continue、break
* `continue` 退出单次循环执⾏下⼀次循环
* `break`    终⽌循环
