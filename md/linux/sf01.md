### CentOS 系统安装 JDK1.8
 
#### 1. 官网下载安装包
![](https://fgq233.github.io/imgs/linux/sf001.png)
 
#### 2. finalshell 使用 root 用户连接系统

#### 3. 创建安装目录
* `mkdir /usr/local/java`
* 使用拖拽方式，将安装包上传到`java`目录

#### 4. 解压安装包
```
cd /usr/local/java
tar -zxvf jdk-8u351-linux-x64.tar.gz
```
